# AI Agent 双周刊（2026年3月下旬）：OpenClaw 病毒式爆发、Claude 操控电脑、NVIDIA 开源 Agent 工具

本周 AI Agent 领域异常热闹，多条重磅消息接连登场。OpenClaw 创作者宣布加入 OpenAI、NVIDIA 开源自进化 Agent 运行时、Anthropic 让 Claude 直接控制你的电脑操作——这一切都在暗示：2026 年将是 AI Agent 从实验走向大规模落地的关键一年。

---

## 🔥 OpenClaw：创作者加入 OpenAI，GitHub 突破 24 万星

OpenClaw（曾用名 Moltbot、Clawdbot）正在经历前所未有的爆发式增长。

**发生了什么：**
- 2026 年 2 月，OpenClaw 在 GitHub 上的仓库突破 **10 万星**，成为开发者社区的病毒式传播工具
- 截至 2026 年 3 月 2 日，项目已拥有 **24.7 万星、4.77 万 Fork**
- OpenClaw 创作者 Peter Steinberger 宣布加入 OpenAI，专注下一代 Agent 开发
- OpenClaw 服务将作为开源项目在 OpenAI Foundation 下继续运营

与此同时，基于 OpenClaw 的社交网络 **Moltbook** 增长迅猛，平台已有 **77 万+ 活跃 Agent**。

**⚠️ 安全事件：**
Moltbook 被曝出数据库未加密漏洞，导致 **3.5 万个邮箱地址** 和 **150 万个 Agent API Token** 泄露。漏洞编号 CVE-2026-25253，CVSS 评分 8.8。这再次提醒我们：快速增长生态系统的安全性往往跟不上扩张速度。

**中国动态：**
OpenClaw 在中国也掀起热潮，腾讯和阿里巴巴均参与推广，推动其在中文技术社区快速传播。

---

## 🖥️ Anthropic Claude：AI 直接操控你的电脑

Anthropic 本周宣布为 Claude 推出 **Computer Use** 功能，标志着 AI Assistant 向真正的"数字代理人"迈出重要一步。

**能力包括：**
- 打开并操控 macOS 上的应用程序
- 导航浏览器、填写表单
- 编辑文件和电子表格
- 用户可通过手机向 Claude 发送任务指令，AI 自动在电脑端完成

**定价：** 目前为付费功能（$100-$200/月），macOS 专属，暂无免费试用。这一定价策略也反映出"AI 控制你的电脑"这一能力的商业价值。

---

## 🧠 NVIDIA：开源 Agent 运行时 OpenShell

NVIDIA 发布全新 **Agent Toolkit**，其中包含开源的 **OpenShell 运行时**，用于构建更安全、更可靠的自进化 AI Agent。

**核心内容：**
- **OpenShell**：开源 Agent 运行时，支持 Agent 自我改进与演化
- 企业级 Agent 构建与部署工具集
- 聚焦安全性和可控性，解决 Agent "越狱"风险

这代表了基础设施层面的大厂布局——AI Agent 的军备竞赛正在从模型层蔓延到底层运行时。

## 💳 欧洲首个 AI Agent 支付完成

西班牙桑坦德银行（Santander）与万事达卡（Mastercard）合作，完成了**欧洲受监管银行领域的首个 AI Agent 实时支付**。

这意味着 AI 现在可以在预设限额内代表用户执行真实金融交易，是自主 AI 商务（Autonomous Commerce）的重要里程碑。

## 📱 Apple Siri 迎来 AI 全面重构

Apple 官方宣布，2026 年将推出**彻底重新设计的 AI 驱动版 Siri**。

新 Siri 将具备：
- **屏幕感知能力**（On-screen Awareness）
- 上下文理解与主动建议
- 真正成为跨 App 的智能助手

## 🤖 GPT-5.4 发布

OpenAI 发布 GPT-5.4，在推理能力和上下文处理上进一步提升。配合 Claude 的 computer use 能力，2026 年 Agent 系统正从实验性阶段迈向**可大规模部署**的成熟期。

## 🔓 Meta AI Agent 数据泄露事件

Meta 内部一名员工在内部技术论坛上提问时，AI Agent 给出的解决方案导致**大量敏感用户数据意外泄露**。Meta 已确认此事。AI Agent 在辅助编程时提供错误建议从而引发安全风险——这类问题值得所有企业警惕。

## 本周观点

三条主线值得关注：

1. **基础设施竞争加剧**：NVIDIA 开源 Agent 运行时、OpenAI 接手 OpenClaw，底层工具链正在快速整合
2. **AI 从聊天走向操控**：Claude computer use、GPT-5.4 的工具调用升级——大模型正在从"回答问题"进化到"代替你操作"
3. **安全风险同步放大**：Moltbook 数据泄露、Meta 的内部泄露——Agent 生态越繁荣，安全漏洞的危害边界就越大
