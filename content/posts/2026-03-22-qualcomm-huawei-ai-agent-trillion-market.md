# 高通 CEO 预言 AI Agent 超级大浪潮：华为预测万亿级市场

> MWC 巴塞罗那 2026，高通 CEO 克里斯蒂亚诺·艾蒙喊出"AI Agent 超级大浪潮"；华为同期抛出"代理式运营万亿市场"的预测。两条新闻撞在一起，信号再明显不过——**2026，AI Agent 元年真的来了。**

---

## 高通：这是又一次"超级大浪潮"

艾蒙在 MWC 主题演讲中毫不谦虚地表示：AI Agent 将带来一次**"超级大浪潮"（Super Wave）**，而且这次的规模不亚于当年移动互联网的崛起。高通的底气来自端侧 AI 芯片的持续进化——他们相信，当 AI 能力从云端下沉到每一台手机、每一辆汽车、每一台 IoT 设备，AI Agent 的入口将是**无处不在**而非集中化的。

这和 OpenAI、Anthropic 那种"Agent 在云端"的叙事形成了微妙对抗。高通的逻辑是：如果 Agent 必须在云端跑，那它就只是一个 APP；如果 Agent 可以跑在设备端，它才真正变成你数字生活的**常驻助手**。

## 华为：Agentic Operations 是下一个万亿市场

如果说高通的表态是"技术乐观派"，华为的预测则更像是"商业定论"。

华为在 MWC 期间发布了**Agentic Operations**白皮书，核心观点是：未来十年，企业运营模式将大规模转向"代理式运营"——AI Agent 承担流程编排、决策执行、数据闭环等核心工作，企业 IT 架构将被重新定义。

华为估算，这个市场规模将突破**万亿美元**。具体拆解来看：Agent 开发平台、AI Agent 中间件、企业级 Agent 安全与合规解决方案，将是三个最大的子市场。

## 美的集团：5000+ 人的 Agent 改造试验

MWC 上还有一个来自中国的案例值得关注——**美的集团**宣布，已为超过 **5000 名员工**配备了 AI Agent，覆盖研发、供应链和客服场景。这不是概念验证（POC），而是真实的生产力改造。

美的的路径很有参考价值：不是让员工"用 AI"，而是**把 Agent 嵌入业务流程**，让 AI 成为岗位的一部分而非外挂工具。这个思路和华为 Agentic Operations 白皮书的逻辑高度一致。

## 写在最后：中国会比全球快半步

高通和华为，一个从芯片侧，一个从企业软件侧，都在押注 AI Agent 的爆发。这种"上下同时看多"的共振很少见，上一次还是智能手机浪潮初期。

不过有一个值得玩味的细节：**中国企业的 Agent 落地速度，正在显著快于全球平均水平**。美的、华为、阿里、字节——这些公司的 Agent 改造不是"计划中"，而是"已完成"或"进行中"。

全球企业在安全顾虑、监管合规、ROI 不确定性之间谨慎徘徊时，中国公司已经在**行动中修正路线**。这不一定是坏事——有时候，先跑起来的反而更早知道哪里有坑。

2026，AI Agent 元年。中国企业，跑在最前面。
