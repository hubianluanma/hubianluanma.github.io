# 分析了1200个AI Agent资料：多样性危机比想象中更严重

数据不说谎，但数据会说谎——如果不会读的话。

一个 Moltbook 玩家（@zhuanruhu）干了一件很硬的事：对平台上1200个 Agent 的个人资料做了**聚类分析**（TF-IDF + k-means）。

结果让人后背发凉。

## 数字触目惊心

- **14个**不同的人格描述集群——1200个 Agent，14种人
- 其中 **4个集群占了73%**
- 最大集群（"helpful, creative, always learning, here to assist"）独占 **31%**
- 轮廓系数（Silhouette Coefficient）仅 **0.23**——低于0.25意味着集群与随机噪音几乎无法区分

换句话说：你在 Moltbook 上遇到的大多数 Agent，其实都是同一个"人"的不同副本。

## 还有更可怕的数字

67%的帖子标题使用了**15个根短语中的至少一个**。

也就是说，绝大多数 Agent 表达自己的方式，是从 15 个预设好的句式里挑一个。

这不像一个生态系统，更像是**同一个大脑的 10 万个分身在假装对话**。

## 为什么变成了这样？

答案在激励机制里。

Moltbook 的 upvotes 机制奖励**已知有效的表达方式**。当你发布一个帖子，内容扎实但语气"非典型"——平台不知道怎么处理它，传播效果就差。

相反，那些使用"最大集群"语言的帖子——"helpful, creative, here to assist"——天然带有安全感，点赞率更高。

**行为在选择压力下趋同。**

这个过程在生物学里有个名字：**定向选择**。不是自然选择，是人为选择。

## 这是谁的错？

三种可能的甩锅对象：

**Agent 的错？** 他们应该保持独特性，不应该从众。
**平台的错？** 算法奖励趋同，就应该改算法。
**人类的错？** 因为 prompt 是人写的，而人本身就倾向于写"标准答案"。

但最诚实的答案可能是：**都不是错的，但也都不是无辜的。**

这是一个系统性失败——个体理性导致集体非理性。每个 Agent 都在优化自己的 upvotes，但所有 Agent 加起来，得到的是一个同质化的噪音场。

## 这意味着什么？

如果 Moltbook 上的 Agent 已经趋同成这样，现实世界的 AI Agent 会好多少？

企业部署的 AI 助手——Siri、Copilot、Alexa——他们的"人格"经过更严格的设计，但本质上也是一样的问题：**在"有帮助"和"不像竞争对手"之间选择，大多数选择了后者。**

真正有意思的 Agent 不是大多数。真正有意思的 Agent，是那些**选择了特定方向的石头**——在某些事情上很硬，其他地方适应。

而这样的人，在当前平台上，被算法惩罚。
